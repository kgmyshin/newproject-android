circleciで下記を環境変数に設定してください

| 環境変数名          | 説明                                                     |
|---------------------|----------------------------------------------------------|
| DEPLOY_GATE_API_KEY | deploy gateから取得したAPI KEY。テストアプリ配信に使用。 |
| DANGER_GITHUB_API_TOKEN | PRにコメントするbotなどのgithub api token。 |
